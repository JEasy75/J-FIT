# DHA-50 Base Camp

Static iPhone-friendly training log.

## GitHub Pages

1. Create a GitHub repository.
2. Upload these files to the repository root.
3. Go to Settings → Pages.
4. Source: Deploy from branch. Branch: main. Folder: /root.
5. Open the published URL in Safari on iPhone.
6. Share → Add to Home Screen.

## Important

The workout logs are stored in the browser/device via localStorage. Use Export Data inside the app before clearing Safari data or switching devices.
